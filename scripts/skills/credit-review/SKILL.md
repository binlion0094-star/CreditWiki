# 银行信贷审查聚合查询技能 (Credit Review Skill)

## 技能概述
> 本技能用于银行信贷审查场景，聚合查询企业多维度信息，自动生成结构化信贷审查报告。

## 使用方式
用户输入企业名称或统一社会信用代码，触发本技能，自动执行全维度查询。

## 触发词
- "信贷审查"
- "企业信用报告"
- "贷前调查"
- "企业分析"
- "授信分析"

---

## 查询维度（8大模块）

### 1. 企业基本信息
- 工商登记信息（名称、注册资本、成立时间、经营范围）
- 统一社会信用代码
- 登记状态（存续/吊销/注销）
- 注册地址与实际经营地址
- 法定代表人

### 2. 公司治理结构
- 股权结构（股东列表、持股比例）
- 董事会/监事会组成
- 关联企业识别（母子公司、兄弟公司）
- 公司章程摘要

### 3. 实控人与股东信息
- 实际控制人身份、背景
- 控股股东基本信息
- 股东穿透图（受益所有权链）
- 股东信用记录/涉诉情况

### 4. 主营业务与行业分析
- 主营业务描述
- 行业分类（国民经济行业分类）
- 行业地位（细分领域排名）
- 行业政策环境（支持/限制类）
- 上下游产业链分析
- 竞争格局

### 5. 银行贷款融资信息
- 存量贷款（银行、金额、期限、担保方式）
- 信用评级（如有）
- 对外担保情况
- 未结清信贷余额汇总
- 民间融资/债券/信托情况

### 6. 企业财务分析
- 资产负债表关键指标
- 利润表核心数据
- 现金流量表摘要
- 关键比率分析：
  - 盈利能力：毛利率、净利率、ROE、ROA
  - 偿债能力：流动比率、速动比率、资产负债率
  - 营运能力：应收账款周转、存货周转
  - 发展能力：营收增长率、净利润增长率
- 与行业均值对比

### 7. 授信方案建议与政策匹配度
- 建议授信额度（理论值计算）
- 建议授信期限
- 建议担保方式（抵押/质押/信用）
- 贷款用途合理性
- 政策匹配度评分：
  - 是否属于优先支持类行业
  - 是否符合我行信贷政策
  - 环保/能耗/产能合规性
  - 宏观审慎评估分类

### 8. 审查结论
- 综合评分（1-10分）
- 风险等级：低/中/高
- 明确意见：推荐/有条件推荐/不推荐
- 主要风险点（列示）
- 贷后管理建议

## 数据来源优先级
1. **Tushare Pro** → 上市公司/发债企业财务数据、股票行情
2. **新浪财经API** → 实时行情、公告
3. **工商数据** →（需第三方接口，如天眼查API，此处预留接口位）
4. **本地知识库** → 行业研究报告、监管政策文件

## 输出格式
最终输出为结构化 Markdown 报告，包含：
- 封面（企业名称、报告日期、审查人）
- 8大模块正文
- 风险提示
- 签章区
