## 行业范围

消费品涵盖：食品饮料（白酒/乳制品/调味品/休闲食品）、家电（白电/厨电/小家电）、纺织服装、日化用品、汽车4S店等。

## 行业核心逻辑

```
消费品核心竞争力 = 品牌力 × 渠道力 × 产品力
                  ↓          ↓         ↓
              定价权      覆盖率    更新频率
```

## 核心风险特征

| 风险维度 | 主要表现 |
|----------|----------|
| 周期风险 | 汽车/家电受地产链影响，食品饮料相对抗周期 |
| 库存风险 | 经销商压货导致渠道库存积压 |
| 现金流风险 | 白酒/家电账期长，经营现金流与利润背离 |
| 竞争风险 | 格局分散，价格战压缩利润 |
| 食品安全/产品安全 | 黑天鹅事件冲击品牌 |

## 重点细分行业审查

### 白酒

- **经销模式**：先款后货为主，但大商制下账期较长
- **库存风险**：渠道库存周期（茅台约1个月，五粮液1-2个月）
- **高端酒护城河**：品牌壁垒极强，现金流优异
- **看预收款**：预收款下降往往是景气下行先导指标

### 乳制品

- **原奶成本波动**：生鲜乳价格周期影响毛利率
- **伊利/蒙牛双寡头格局**：中小乳企竞争激烈
- **常温奶格局稳定，低温奶增速快**：产品结构升级机会

### 家电

- **地产后周期**：厨电/空调与地产竣工相关性高
- **出口代工风险**：代工企业议价能力弱，毛利率低
- **618/双11**：大促期间压货，经销商资金压力大

### 汽车4S店

- **库存融资**：汽车厂家金融公司提供担保，银行敞口在经销商信用
- **品牌分化**：豪华品牌（ABB）盈利好，自主品牌亏损面大
- **新能源转型**：新势力直营模式冲击传统4S店模式

## 财务审查重点

```
核心检查项：
1. 渠道库存（是否异常积压）
2. 应收账款账龄（品牌商对经销商的信用政策）
3. 经营现金流 vs 净利润（利润含金量）
4. 经销商杠杆倍数（库存融资依赖度）
5. 现金周期（存货周转+应收账款周转-应付账款周转）
```

## 关键指标参考

| 子行业 | 核心指标 | 健康值 |
|--------|----------|--------|
| 白酒 | 渠道库存（月） | <2个月 |
| 乳制品 | 原奶成本占比 | <40% |
| 家电 | 出口占比 | <30%（规避汇率+贸易风险） |
| 汽车4S | 单店盈利 | >500万/年 |
