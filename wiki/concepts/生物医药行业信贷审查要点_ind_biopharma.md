## 🔬 第一性原理分析

### 核心规律
【生物医药行业信贷】的核心规律：医药行业是"政策定价+研发赌注"的双重游戏——集采把仿制药价格打到地板让存量业务失去利润弹性，创新药研发成功率不足10%且商业化周期长达10年，银行信贷进入这个行业本质上是在赌"哪个药企能做出差异化竞争力"，而集采常态化和医保谈判让这个赌注的风险收益比越来越差。

### 周期特征
- **政策强周期**：集采每轮都重新洗牌，价格降幅超预期
- **研发长周期**：创新药从立项到上市10年+，商业化前无现金流
- **分化加剧**：仿制药企艰难转型，创新药企享受溢价

### 银行信贷关联
- 固定资产投资（GMP改造、产能扩张）
- 研发费用垫资（临床试验费用）
- 供应链金融（商业化阶段的药品经销商）
- 并购贷款（行业整合机会）

## 📊 行业分类体系

### 按商业模式
| 类型 | 特征 | 银行介入时机 |
|------|------|------------|
| **仿制药** | 集采竞价、利润薄 | 商业化阶段、账期管理 |
| **仿创药（Me-better）** | 差异化仿制，有一定定价权 | 临床后期确定性高时 |
| **创新药（First-in-class）** | 高风险高回报 | 投资期不适合信贷 |
| **中药** | 政策保护力度大 | 独家品种=稳定现金流 |
| **疫苗** | 高门槛、政策免疫 | 大品种上市后 |

### 产业链
```
研发 → 临床试验 → 注册申报 → 生产（GMP） → 商业化 → 销售
 ↑                                                   ↓
 └──────────── 资金来源（VC/IPO/贷款） ←─────────────┘
```

## 📋 信贷审查核心指标

### 仿制药企业
| 指标 | 说明 | 警戒线 |
|------|------|--------|
| 集采中标率 | 中标=销量保障但价格低 | 未中标=存量市场流失 |
| 毛利率 | 集采前>70%，集采后可能<30% | 持续下降 |
| 应收账款周转 | 医院回款周期6-12个月 | 账龄恶化 |
| 品种梯队 | 是否有下一批接续品种 | 青黄不接 |
| 出口业务 | 是否开拓国际市场对冲 | 认证周期长 |

### 创新药企业
| 指标 | 说明 |
|------|------|
| 临床进度 | II期/III期成功率差异大 |
| 资金储备 | 能撑几个临床（账上现金/月烧钱） |
| 商业化能力 | 自建销售团队 vs 代理 |
| 医保谈判 | 进入医保=销量保障但价格折扣大 |

## ⚠️ 核心风险因素

### 集采风险（最高权重）
- 2025年第十批国采平均降幅超70%
- 未中标=失去主流市场，中标=牺牲利润
- 存量和增量双重挤压

### 研发风险
- 创新药研发成功率：临床I期10%，II期30%，III期50%
- 商业化不确定性：即使上市也可能销售不及预期
- 靶点扎堆：同靶点多家竞争，医保谈判内卷

### 政策风险
- 医保谈判年压价
- 合理用药监控
- 反垄断调查（原料药）

- 典型仿制药企转型困境
