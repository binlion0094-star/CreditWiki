## 行业范围

港口（集装箱/干散货/液体散货）、航运（集装箱/干散货/油轮/液化气LNG）、船舶制造、港口物流、综合物流服务商。

## 行业核心逻辑

```
港口航运 = 国际贸易晴雨表 + 大宗商品周期
                 ↓
          需求端：全球贸易量、货物结构
          供给端：运力投放节奏（决定周期长度）
```

## 周期特征

**强周期行业**，周期长度约5-8年（造船周期2-4年决定运力错配）

| 周期阶段 | 运价表现 | 典型盈利 | 信贷风险 |
|----------|----------|----------|----------|
| 复苏期 | 运价反弹 | 扭亏 | 中 |
| 繁荣期 | 运价高位 | 暴利 | 低（但需防拐点） |
| 衰退期 | 运价暴跌 | 亏损 | 高 |
| 萧条期 | 运价底部 | 持续亏损 | 极高 |

## 核心风险点

### 一、运力过剩风险（当前需重点关注）

- 2020-2023年集装箱船大量下水（订单潮），2024-2026年集中交付
- 散货船新增运力同样面临过剩压力
- **审查手持订单/现有运力比**：>30%为高风险警示

### 二、地缘政治与贸易摩擦

- 红海危机改道（2023年底至今）：短期利好即期运价，长远增加成本
- 中美贸易摩擦：货主结构变化，影响枢纽港地位
- 碳排放约束（IMO 2050）：老旧船舶加速淘汰，资本支出压力

### 三、港口竞争格局

- **枢纽港 vs 喂给港**：枢纽港腹地广阔、议价能力强
- 区域内港口同质化竞争激烈，压装卸费费率
- 港口整合趋势（省级港口集团整合）

### 四、航运企业财务特征

```
高负债 + 高波动 + 轻资产运营
特点：
- 银行借款购置船舶（抵押物：船舶）
- 经营租赁下巴租船（经营杠杆极高）
- 现金流的季节性和年度波动性大
```

## 船舶抵押物特殊性

| 船型 | 价值评估难点 | 抵押率建议 |
|------|--------------|------------|
| 集装箱船 | 受运价周期影响大 | 50-60% |
| 干散货船 | 价格透明，有波罗的海指数参考 | 55-65% |
| 油轮 | 受原油价格联动 | 50-60% |
| LNG船 | 技术门槛高，期租率稳定可稍高 | 55-65% |

## 财务审查重点

```
核心检查：
1. 期租比例（锁定收益比例越高越稳定）
2. 船舶账龄与船龄（老龄船维修成本高，CII碳税压力）
3. 有息负债/总资产（航运企业普遍高杠杆，>70%为高危）
4. 美元敞口（收入美元为主，成本RMB，汇率风险大）
5. 燃油成本占比（一般占运营成本30-40%）
```

## 关键指标参考

| 指标 | 健康 | 预警 |
|------|------|------|
| 期租锁定比例 | >60% | <30% |
| 手持订单/运力 | <20% | >30% |
| 船龄加权 | <10年 | >15年 |
| 美元敞口对冲比例 | >70% | <50% |
