**2025年度个人述职报告**

# 2025年个人述职报告（授信审批部总经理陈斌兵）

2025年，在分行党委的领导下，授信审批部围绕“**专业赋能**”与“**标准化管理**”两大主线开展工作：“赋能”为矛，旨在让资产业务高质量奔跑，在发展中破局、在创新中前行；“标准化管理”为盾，是底座、是根基，是行稳致远的前提与基础。基于两大工作主线，授信审批部积极有序开展工作。现将全年工作汇报如下：

1. **党风廉政建设情况**

# 本人坚守入党誓言，时刻牢记党员身份。作为分行授信审批部负责人深知责任在肩，始终坚持“一岗双责”，将业务发展与风险防控、职责履行与廉政建设同部署、同落实。

# 在授信审批工作中，严格执行信贷政策与审批流程，严把准入关，精准研判项目风险，助力实体经济高质量发展；同时，带头严守廉洁纪律，强化部门全员合规意识，常态化开展警示教育，筑牢拒腐防变思想防线。组织多场专题培训，牵头开展信审提质增效攻坚行动，如全员参与普惠业务审查、优化简化审查报告与业务流程等，效率提升10%以上。

# 以责履事，以廉立身，做到业务工作与党风廉政建设两手抓、两手硬，为分行信贷资产安全与稳健运营保驾护航。本人始终坚持以习近平新时代中国特色社会主义思想为指导，发挥基层党支部战斗堡垒作用。组织支部第一议题学习10个、党委委员与支部书记讲党课3场，并开展与石门二路街道华沁社区党总支、邮储银行长宁区支行党总支的党建共建活动。在作风与合规建设方面，严格贯彻中央八项规定精神学习教育，组织到中共二大会址参观学习“清风正气满乾坤——党章中的作风建设”专题展览、廉政警示教育4次及廉洁授信主题党课1次。在组织建设方面，今年发展正式党员一名。

1. **关键绩效（KPI）指标完成情况**

# 资产质量指标：截止2025年11月，上海分行资产质量一类行排名第一。

# 审查质量指标：截止2025年11月，总行授信审批部对上海分行送审质量考评为一类行第二名。

# 业务量指标：全年共计审批三大条线业务8696笔，合计审批敞口18020亿元（企金、零售、金融市场），累计放款16253笔，放款金额6617亿元。

# 其中，审批端：企金1545笔，敞口7902亿元；小企业1976笔，敞口108亿元；零售4869笔，敞口222亿元；金融市场306笔，敞口9788亿元。

# 放款端：对公7789笔，金额6319亿元；零售8464笔，金额299亿元。

1. **重点工作开展情况**

# （一）专业赋能：在发展中解决问题

# 过去一年，带领分行授信审批部以“发展中解决问题”为行动纲领，实现了三个关键维度的突破：在“勇担当”中化解存量风险，卸下存量包袱；在“敢创新”中构建增量资产，布局未来赛道；在“谋长远”中推动资产结构转型，夯实高质量发展根基。

# 1、积极推动重大化解项目。2025年是攻坚克难的一年，地产行业持续触底下探，分行资产质量面临重大挑战。授信审批部主动担当、积极作为，牵头多笔风险项目化解工作，深入一线调查掌握项目风险，一户一策制定化解处置方案，最终推动多笔重大项目的成功化解，包括绿地董家渡90亿元、圆泉静安moho项目48.96亿元、经纬置地存量20.5亿元重组、广汇汽车9.89亿元，全年合计完成大额风险项目重组项目13笔，金额214.64亿元。

# 2、助推大类资产构建。带领授信审批部以专业为抓手，积极探索新的价值增长极，通过深入研判市场趋势，勇于在并购业务、固定资产贷款、资本市场业务等领域进行大类资产构建，审批了KKR收购大窑、赛斯潘置换船舶股东资产贷款、秦淮出售竞标项目、星巴克中国公司出售竞标项目、安能物流私有化项目等多笔市场明星项目，合计审批项目类资产95笔，合计金额1609亿元，落地金额406.68亿元。不仅助力分行构建高收益长久期的信贷资产，打开新的发展空间，也在市场上擦亮我行“投资银行”专业名片，培育了面向未来的核心竞争力。

# 3、优化资产结构布局。过去资产结构中存在“基础类客群占比不足”“头大腰细”问题，是影响整体发展稳定性和抗风险能力的深层隐忧。授信审批部今年将工作重心放在培育“中型客户”与“上市公司”客群上。年初推行的上市公司主动授信，合计审批主动授信客户47户，敞口金额137亿元，助力上市公司授信覆盖率由24%提升至33.7%。“中型客户”方面，2025年合计审批中型客户346户，金额188.29亿元，户均审批额度5400万元。中型客户增量84户，额度增长47亿元，笔数增长率32%，额度增长率33%。客群的多元化，帮助分行资产结构从“依赖少数大客户”向“大中小客户协同互补”的健康生态积极转型，为分行长远高质量发展奠定了更为坚实与均衡的基础。

# （二）标准化管理：明确标准、规范作业、精准管控

# 1、立标尺：明确普惠业务审查放款的关键标准。2025年，牵头梳理了普惠业务的审查及放款要求，通过《关于进一步明确现阶段普惠小微审查要求的通知》、《关于普惠业务放款操作标准化的要求及近期有关业务的风险提示》等一系列文件，明确授信客户的准入标准及条件及关键性的审核标准、明确普惠放款的关键性要求，包括决议的规范性、核保核签核担保手续的规范性、贷款用途的合规性、系统操作的规范性等。通过明确标准，大幅提升营销精准性，退件率由上年的17%下降至今年的11%，审批通过率提高了6个百分点。

# 2、重执行：严格作业流程节点管控。加强流程节点管控，大企业方面，执行“上报即开卷”要求，制定明确的流程时间节点，受理项目T+3开卷，T+5下发补充，提升流程流转的标准化、透明性。普惠业务方面，创新性提出“企金普惠小微客户续作类业务简化评审方案”，配套简化审批流程。针对已设置风险合规部的区域中支，执行续作简化流程，由区域支行风险合规部完成续作方案审定意见，经分行授信审批部复核无误后，适用绿色通道直接报对应审批官完成审批。该机制下发后，大企业的审查时效由2.12笔/周提升至2.59笔/周，审查提速达20%；企金普惠平均审结时效7.84个工作日，在人员配置偏紧的情况下，较上年进一步压缩0.56个工作日。

# 3、促效能：实施差异化分层与精准管控。在统一标准的基础上，授信审批部着力推动“标准化”向“精细化”演进，根据客户的主体资质、行业归属、经营现状、信用风险与收益匹配度，综合评估划分为积极合作（A类）、鼓励合作（B类）、一般合作（C类）、审慎合作（D类），对于风险特征清晰、模式简单的标准化业务（如部分优质国企、上市公司等），创新推出简化的审查报告模板与审批流程，建立了高效的“快速通道”，将有限的管理与风控资源，从大量重复性劳动中解放出来，得以更聚焦地投入到复杂项目、创新业务及重点客户的价值挖掘与深度评审中。实现了 “简单业务高效通过，复杂业务精耕细作” 的管理目标。

# （三）引入AI大模型，提升风险评审的数智化水平。

# 随着金融监管政策的不断演变，传统的法规查询方式已无法满足当前的业务要求，亟需引入一套涵盖各类金融法规及主要监管机构政策的数据库，以提升业务人员的法规获取效率和准确性。2025年授信审批部联合信息科技部申请引入AI金融法规数据服务，用于信审工作中业务送审、项目评估、放款下柜、尽调等环节，搭建AI智能金融法规知识库，全面覆盖九千多部现行有效法规，提供智能检索、法规关联及合规建议。

1. **存在不足以及改进措施**

（一）存在不足

作为分行授信审批部负责人，对照岗位职责，深刻反思自身履职尚存在不足，在一定程度上制约了部门工作质效，具体如下：

1、前瞻统筹规划不够有力，产能弹性不足。未能前瞻性预判业务周期波动，人力与流程配置缺乏系统规划，部门长期处于紧平衡刚性状态，未建立动态产能管理机制。面对业务高峰期，无弹性调度与应急方案，仅靠 “全员审查” 临时应对，暴露出应急管理短板，影响服务体验与市场响应速度。

2、创新驱动不力，数智化转型滞后。对数字化转型战略意义认识不深，推动力度不足。智能化工具应用停留在初步阶段，线上自动化流程覆盖率低，内外部多源数据未有效整合，“数据孤岛”问题突出。未能借科技手段解放人力、提升风险识别精准度，数字化转型与业务发展脱节。

3、协同赋能不够，跨部门联动薄弱。在政策传导与前端赋能上投入不足，未建立高效协同体系，导致前线部门对审查标准理解不深，材料补报、重复上报问题频发。对前端尽调指导约束不足，部分业务信息缺失或 “选择性披露”，审查部门被迫承担额外尽调工作，浪费资源且降低审批效率。

4、管理精细不足，流程管控薄弱。缺乏精细化管理思维，未明确各环节责任边界，流程衔接存在 “模糊地带”，任务交接依赖人工，无自动化节点控制与预警机制，易出现滞留遗漏。时效管理仅停留在事后统计，无实时穿透式监控，难以精准定位延误堵点，部门运营效能未充分释放。

**（二）改进措施**

针对此前履职中暴露的统筹不足、数智化滞后、协同薄弱、管理粗放等问题，结合总行授权优化契机，我将牵头聚焦三大方向推进整改，以实干补短板、促提升：

1、锚定政策导向，精准校准资产构建方向。紧扣总行新版授权要求，牵头推动客群、产品、产业、领域四维发力。聚焦腰部客群与优质主体，助力供应链、先进制造业等重点客群拓展；强化固定资产贷款、贸易融资等产品赋能；深耕船舶产业链，打好产品组合拳；优化并购贷款等资产结构，坚守合规底线，切实破解重点客群触达率低、资产转化率不足的问题。

2、推动角色转型，强化风险赋能实效。以审查趋前作业打破被动局面：一是优化评审资源保障重点项目，实现全流程跟进；二是构建重点行业图谱，提供风险地图，推动审查从 “看企业” 到 “看产业生态” 跃升；三是建立前瞻会商机制，在营销初期介入，以行业对标、风险优化建议引导业务设计，从 “风险把关者” 转变为 “价值共创伙伴”，强化对前线的政策传导与赋能。

3、深化内部治理，系统性提升运营效能。针对产能与管控瓶颈，从三方面发力：一是流程提效，推出标准化产品快速通道，实行清单式审查、模板化报告，释放人力弹性；二是过程可控，绘制审查流程全景图，设置关键节点打卡与超时预警，实现实时穿透式管控；三是文化筑基，树立专业标杆，开展跨部门复盘与审查报告共享，筑牢团队专业能力与协同意识，全面提升审查效率与服务体验。
